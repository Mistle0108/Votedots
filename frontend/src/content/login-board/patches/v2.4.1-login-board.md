# 로그인 전 안내 게시판 구조 추가

## 추가 내용
- 로그인 화면에서 최근 패치 내역과 로드맵을 함께 볼 수 있는 읽기 전용 게시판 구조를 준비했습니다.
- 게시판과 로드맵은 상단 탭으로 전환할 수 있게 구성했습니다.
- 상세 내용은 Markdown 문자열을 렌더링하는 방식으로 맞췄습니다.

## 운영 방식
- 게시글은 버전 또는 주제별로 Markdown 파일을 추가합니다.
- 로드맵은 분기별 그룹 안에 항목을 추가하는 방식으로 확장합니다.

`Markdown` 본문은 추후 백엔드에서 문자열로 내려주는 형태로 바뀌어도 같은 렌더링 흐름을 유지할 수 있습니다.
